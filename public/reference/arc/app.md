# `@app`

## `@app` is where you declare your app namespace

### Syntax
- Lowercase alphanumeric string
- Maximum of 10 characters
- Dashes allowed; underscores not allowed
- Must begin with a letter

### Example
For example, to create the app "foobaz":

```arc
@app
foobaz
```
